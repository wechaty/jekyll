SpringBoot搭建wechaty智能聊天机器

smwsk Follow Aug 17, 2020 · 1 min read

本项目是基础wechaty项目实现微信的聊天功能、通过微信对话开放平台提供的API进行智能的应答、实现了微信的智能聊天、最后非常感谢wechaty的无私奉献。

智能机器人功能

  • 高级转人工
  • 周边生活服务
  • 聊聊电影
  • 讲笑话
  • 成语接龙
  • 天气查询
  • AI时光机
  • 心里年龄测试
  • 中英互译功能
