з®ҖеҚ•иҒҠиҒҠjavaзүҲPuppet Managerзҡ„е®һзҺ°

redmaple1 Follow Jun 16, 2020 В· 1 min read

жҲ‘д»¬жқҘз®ҖеҚ•иҒҠдёҖиҒҠjava-wechatyдёӯPuppet Managerзҡ„е®һзҺ°гҖӮ

еүҚиЁҖ

вҖӮвҖӮвҖӮвҖӮйҰ–е…ҲпјҢиҝҷйҮҢеҖҹз”Ёjava-wechatyзҡ„maintainerзҠҖеҲ©иұҶзҡ„еҚҡе®ўд»Ӣз»ҚдёҖдёӢwechatyжҳҜд»Җд№ҲпјҢд»ҘеҸҠjavaзүҲжң¬зҡ„еүҚдё–д»Ҡз”ҹгҖӮ

зҠҖеҲ©иұҶ з»ҲдәҺжңүдёҖдёӘJavaеҸҜд»Ҙз”Ёзҡ„еҫ®дҝЎжңәеҷЁдәәдәҶ

иғҢжҷҜ

вҖӮвҖӮвҖӮвҖӮеҸӮдёҺејҖеҸ‘java-wechatyжңүдёҖдёӘеӨҡжңҲзҡ„ж—¶й—ҙпјҢеңЁејҖеҸ‘зҡ„иҝҮзЁӢдёӯпјҢдёҚе…ҚиҰҒиҝӣиЎҢиҮӘжөӢгҖӮеңЁеүҚжңҹжІЎжңүtokenи°ғиҜ•дёҚдҫҝзҡ„жғ…еҶөдёӢпјҢе°ұжғіеҖҹеҠ©еҚ•жөӢеҜ№жүҖеҶҷд»Јз ҒиҝӣиЎҢйӘҢиҜҒпјҢдҪҶжҳҜжҲ‘еҸ‘зҺ°жғіиҰҒиҝӣиЎҢеҚ•жөӢд№ҹдёҚжҳҜдёҖд»¶е®№жҳ“зҡ„дәӢжғ…гҖӮдёҺд»ҘеҫҖ java web ејҖеҸ‘дёҚеҗҢпјҢжІЎжңүSpringе°ҒиЈ…еҘҪзҡ„еёҰжңүдёҠдёӢж–Үзҡ„testгҖӮдәҺжҳҜпјҢжҲ‘еҖҹйүҙtsзүҲжң¬wechatyзҡ„mockжЁЎеқ—пјҢе®һзҺ°дәҶjava-wechatyзҡ„mock puppetпјҢдё“й—Ёз”ЁдәҺжөӢиҜ•wechatyдёҠеұӮд»Јз ҒйҖ»иҫ‘гҖӮејҖеҝғең°е®һзҺ°е®ҢжҲҗmock puppetд№ӢеҗҺпјҢеҸҲеҸ‘зҺ°дәҶж–°зҡ„й—®йўҳгҖӮйӮЈе°ұжҳҜзҺ°жңүjavaзүҲжң¬зҡ„wechatyеңЁеҲқе§ӢеҢ–puppetзҡ„ж—¶еҖҷпјҢеңЁд»Јз ҒдёӯеҶҷжӯ»дәҶhostie puppetпјҢд№ҹе°ұжҳҜеӣҫдёӯзҡ„GrpcPuppetгҖӮ

вҖӮвҖӮвҖӮвҖӮиҝҷж ·зҡ„иҜқпјҢжҲ‘е°ұж— жі•еҲқе§ӢеҢ–mock puppetдәҶпјҢжүҖд»ҘжҲ‘е°ұжҖқиҖғиғҪдёҚиғҪжңүдёҖдёӘmanagerжқҘз®ЎзҗҶpuppetзҡ„е…·дҪ“е®һзҺ°гҖӮжңүдәҶиҝҷдёӘжғіжі•пјҢйӮЈд№ҲеҰӮдҪ•иҝӣиЎҢдјҳйӣ…зҡ„е®һзҺ°е‘ўпјҹжҺҘдёӢжқҘжҲ‘д»¬з®ҖеҚ•жқҘиҒҠиҒҠгҖӮ

е®һзҺ°жҖқи·Ҝ

  • PuppetManagerжқҘз®ЎзҗҶе…·дҪ“зҡ„puppetе®һзҺ°зұ»
  • еңЁwechatyдёӯйҖҡиҝҮи°ғз”ЁPuppetManagerзҡ„е®һдҫӢеҢ–ж–№жі•пјҢеҲқе§ӢеҢ–еҮәйңҖиҰҒзҡ„puppetе®һзҺ°
  • еҸҜдҪҝз”ЁеҸҚе°„жңәеҲ¶еӨ„зҗҶpuppetе’Ңе…·дҪ“е®һзҺ°зұ»

вҖӮвҖӮвҖӮвҖӮдҪҝз”ЁдёҠиҝ°жҖқи·Ҝзҡ„ж”№йҖ пјҢwechatyеҲқе§ӢеҢ–зҡ„жөҒзЁӢе°ұеҸҳжҲҗдәҶдёӢеӣҫзҡ„ж ·еӯҗгҖӮ

вҖӮвҖӮвҖӮвҖӮеҸҜд»ҘзңӢеҲ°пјҢеңЁwechatyзҡ„initPuppet()ж–№жі•дёӯпјҢдёҚеҶҚжҳҜзӣҙжҺҘеҲқе§ӢеҢ–GrpcPuppetпјҢиҖҢжҳҜдҪҝз”ЁдәҶPuppetManagerзҡ„resolveInstance()ж–№жі•жӢҝеҲ°дәҶеҲқе§ӢеҢ–иҝҮзҡ„puppetе®һзҺ°гҖӮ
вҖӮвҖӮвҖӮвҖӮйӮЈд№ҲPuppetManagerе…·дҪ“жҳҜжҖҺд№Ҳе®һзҺ°зҡ„е‘ўпјҹе…¶е®һеҫҲз®ҖеҚ•пјҢзӣҙжҺҘзңӢд»Јз ҒгҖӮ

const val REFLECTION_BASE_PACKAGE = "io.github.wechaty"
class PuppetManager {

    companion object {
        private val log = LoggerFactory.getLogger(PuppetManager::class.java)

        @JvmStatic
        fun resolveInstance(wechatyOptions: WechatyOptions): Future<Puppet> {
            log.info("PuppetManager resolveInstance(${JsonUtils.write(wechatyOptions)})")

            val reflections = Reflections(ConfigurationBuilder().setUrls(ClasspathHelper.forPackage(REFLECTION_BASE_PACKAGE, Thread.currentThread().contextClassLoader)))

            val subTypes: Set<*> = reflections.getSubTypesOf(Puppet::class.java)
            if (subTypes.isEmpty()) {
                throw java.lang.RuntimeException("expect one puppet,but can not found any one.")
            }

            if (subTypes.size > 1) {
                throw RuntimeException("expect one puppet,but found ${subTypes.size}")
            }
            val clazz = subTypes.first() as Class<*>
            val declaredConstructor = clazz.getDeclaredConstructor(PuppetOptions::class.java)
            return CompletableFuture.completedFuture(declaredConstructor.newInstance(wechatyOptions.puppetOptions!!) as Puppet)
        }
    }

}

вҖӮвҖӮвҖӮвҖӮйҰ–е…Ҳе®ҡд№үдәҶдёҖдёӘеҹәзЎҖзҡ„еҢ…и·Ҝеҫ„пјҢеңЁиҜҘи·Ҝеҫ„дёӢдҪҝз”Ёreflectionsеә“пјҢжү«жҸҸжүҖжңүе®һзҺ°дәҶPuppetзҡ„зұ»гҖӮеӣ дёәжҲ‘д»¬еҝ…йЎ»йңҖиҰҒдёҖдёӘpuppetе…·дҪ“е®һзҺ°зұ»пјҢжүҖд»ҘеҪ“жҲ‘д»¬жү«жҸҸдёҚеҲ°д»»дҪ•puppetзҡ„е®һзҺ°ж—¶пјҢдјҡжҠӣеҮәејӮеёёпјҢе‘ҠзҹҘејҖеҸ‘иҖ…жңҹжңӣжңүдёҖдёӘpuppetе®һзҺ°зұ»пјҢдҪҶжҳҜеңЁclasspathдёӢе№¶жІЎжңүжүҫеҲ°гҖӮ

вҖӮвҖӮвҖӮвҖӮе…¶ж¬ЎпјҢеҪ“еүҚжҲ‘д»¬еҸӘж”ҜжҢҒеҚ•ж¬Ўе®һдҫӢеҢ–дёҖз§Қpuppetе®һзҺ°пјҢжүҖд»ҘеҪ“еңЁи·Ҝеҫ„дёӢжү«жҸҸеҲ°еӨҡдәҺ1дёӘpuppetе®һзҺ°зұ»ж—¶пјҢеҗҢж ·дјҡжҠӣеҮәејӮеёёпјҢжҸҗзӨәејҖеҸ‘иҖ…classpathдёӯеӯҳеңЁдёҖдёӘд»ҘдёҠзҡ„puppetе®һзҺ°гҖӮ

вҖӮвҖӮвҖӮвҖӮдёҠйқўдёӨз§Қжғ…еҶөеқҮжңӘжҠӣеҮәејӮеёёзҡ„пјҢиҜҙжҳҺmanagerе·Із»ҸжүҫеҲ°дәҶйңҖиҰҒе®һдҫӢеҢ–зҡ„puppetе®һзҺ°пјҢжҺҘдёӢжқҘе°ұдҪҝз”ЁjavaеҸҚе°„пјҢзӣҙжҺҘе®һдҫӢеҢ–еҚіеҸҜгҖӮ

вҖӮвҖӮвҖӮвҖӮд»ҘдёҠе°ұжҳҜmanagerе®һдҫӢеҢ–puppetзҡ„з®ҖеҚ•е®һзҺ°гҖӮ

еұ•жңӣ

вҖӮвҖӮвҖӮвҖӮйҰ–ж¬Ўеј•е…Ҙmanagerзҡ„зүҲжң¬дёӯпјҢжҲ‘д»¬жҠҠд№ӢеүҚеј•е…ҘеҲ°wechaty sdkдёӯзҡ„hostie puppetе®һзҺ°жӢҝдәҶеҮәжқҘпјҢеңЁsdkдёӯд»…еј•е…Ҙpuppetе®ҡд№үеұӮгҖӮиҝҷж ·пјҢдҪҝз”ЁиҖ…йңҖиҰҒе“Әз§Қpuppetе®һзҺ°пјҢиҮӘе·ұеј•е…ҘеҚіеҸҜпјҢдҪңдёәsdkе°ұдёҚеҶҚе…іеҝғдәҶгҖӮ
вҖӮвҖӮвҖӮвҖӮдҪҶжҳҜиҝҷж ·еҸҲеўһж·»дәҶдҪҝз”ЁиҖ…зҡ„дҪҝз”Ёй—ЁеқҺпјҢдёҚеҰӮд№ӢеүҚд»…еј•е…ҘsdkеҢ…е°ұиғҪдҪҝз”Ёж–№дҫҝпјҢжүҖд»ҘеңЁжҺҘдёӢжқҘзҡ„зүҲжң¬дёӯпјҢжҲ‘д»¬иҝҳжҳҜжү“з®—еңЁsdkдёӯдҪҝз”Ёhostie puppetдҪңдёәpuppetзҡ„й»ҳи®Өе®һзҺ°пјҢеҰӮжһңдҪҝз”ЁиҖ…жңүдҪҝз”Ёе…¶д»–puppetе®һзҺ°зұ»зҡ„иҜүжұӮпјҢйңҖиҰҒжүӢеҠЁеңЁpomдёӯexcludeжҺүhostie puppetзҡ„й»ҳи®Өе®һзҺ°пјҢз„¶еҗҺеј•е…ҘйңҖиҰҒзҡ„puppetе®һзҺ°еҚіеҸҜгҖӮ

вҖӮвҖӮвҖӮвҖӮеҗҺз»ӯеҸҜиғҪдјҡж”ҜжҢҒеӨҡдёӘpuppetпјҢйӮЈд№Ҳmanagerе°ұйңҖиҰҒйҖӮеҪ“зҡ„ж”№йҖ еҺ»йҖӮй…ҚеӨҡдёӘpuppetе®һзҺ°гҖӮиҝҷйҮҢжңүдёҖдёӘз®ҖеҚ•зҡ„и®ҫи®ЎжҖқи·ҜпјҢеңЁpuppetе®ҡд№үеұӮпјҢжҲ‘д»¬еҸҜд»Ҙе®ҡд№үдёҖдёӘmapping()ж–№жі•пјҢиҜҘж–№жі•зҡ„ж„ҸжҖқжҳҜдҪңдёәдёҖдёӘpuppetпјҢжҲ‘йңҖиҰҒеҰӮдҪ•зҡ„жҳ е°„жүҚиғҪеҲқе§ӢеҢ–гҖӮйӮЈд№Ҳе…·дҪ“жҖҺд№Ҳжҳ е°„е°ұдәӨз»ҷеӯҗзұ»пјҢд№ҹе°ұжҳҜе…·дҪ“зҡ„puppetе®һзҺ°зұ»иҮӘе·ұеҺ»е®һзҺ°гҖӮеңЁmanagerдёӯпјҢжҲ‘д»¬еҸӘйңҖиҰҒдҪҝз”Ёpuppetе®ҡд№үеұӮзҡ„mapping()ж–№жі•е°ұеҸҜд»Ҙе®һзҺ°еҜ№еӯҗзұ»зҡ„жҳ е°„еӨ„зҗҶпјҢдҪңдёәmanagerпјҢе№¶дёҚз”Ёе…іеҝғеҪ“еүҚеҲ°еә•жҳҜи°ҒеңЁеҲқе§ӢеҢ–гҖӮдёӢеӣҫжҳҜеӨ§дҪ“зҡ„з»“жһ„гҖӮ

вҖӮвҖӮвҖӮвҖӮжңүдәҶPuppetManagerпјҢжҲ‘д»¬зҡ„java-wechatyе®һзҺ°еҫ—жӣҙеҠ дјҳйӣ…дәҶдёҖдәӣпјҢиҖҢдё”е®һзҺ°дәҶmock puppetпјҢеҗҺз»ӯе°ұеҸҜд»ҘйҖҡиҝҮеҚ•жөӢжҸҗй«ҳд»Јз Ғзҡ„иҙЁйҮҸпјҢжҸҗеҚҮзЁіе®ҡжҖ§пјҢзӣёдҝЎжҲ‘д»¬зҡ„java-wechatyдјҡи¶ҠжқҘи¶Ҡе®Ңе–„гҖӮ
вҖӮвҖӮвҖӮвҖӮеҰӮжһңдҪ еҜ№wechatyж„ҹе…ҙи¶ЈпјҢжҒ°е·§еҸҲжҳҜjava developerпјҢеҜ№java-wechatyжңүиҮӘе·ұзҡ„жғіжі•жҲ–еҜ№зӣ®еүҚзҡ„д»Јз Ғе®һзҺ°жңүд»»дҪ•еҘҪзҡ„е»әи®®пјҢжңҹеҫ…дҪ зҡ„еҠ е…ҘпјҢдёәjava-wechatyиҙЎзҢ®д»Јз Ғ~
