Development Plan

  • Task Requirements Develop a bot based on Wechaty to implement functions such as telling jokes, idiom chain, guessing pictures, guessing song names, checking flights, checking weather, Weibo hot search reminders, etc.; design Logo and image; develop Piggy Bro Back Garden.
  • Implementation Plan Utilize Wechaty features to implement a simple extensible scaffold and modularly implement various functions. Then collect highly available APIs to implement required functions. The Back Garden uses Node.js + Vue development mode to implement a simple mini-game.

Development Timeline
